- 分支的拉取是非常重要的操作，千万不要一个 master 直接梭哈到底。尤其是很多伙伴，写代码的时候，一个功能验证通过，但后面的一堆流程开发后，代码跑不了了。也不知道从哪出的问题。
- 持久化对象、mapper xml 编写，会有一些 idea 操作的技巧。另外像大厂开发中，并不希望研发通过自动生成的代码进行使用，因为自动生成会有很多无用的代码，有时候有些sql的操作，太通用性也不行，容易出事故。
